# 2. Привет, Мир!

По традиции начнем с программы *Hello, World!*. Эта программа будет выводить на экран текст:

```javascript
Hello, World!
```

Чтобы вывести что-то на экран, нужно дать компьютеру специальную команду. В языке TypeScript такая команда — `console.log()`.
