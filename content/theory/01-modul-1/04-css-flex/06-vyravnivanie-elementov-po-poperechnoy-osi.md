# Выравнивание элементов по поперечной оси

Выровнять элементы по поперечной оси — одна из классических «болей» вёрстки. Способов поставить блок по центру насчитываются десятки; у каждого свои плюсы и минусы, и универсального решения не было.

Flex закрывает задачу одним свойством **align-items**. Не нужно знать точные размеры внешнего и внутреннего блока, считать строки текста или полагаться на перенос — достаточно одного правила (поддержку старых браузеров, вроде IE7, сейчас обычно не учитывают).

## align-items: stretch

Значение по умолчанию для дочерних flex-элементов. Высота подстраивается под высоту контейнера.

Раньше для такого эффекта нередко писали JS. Автовысота сработает, только если у элемента не задана своя высота в CSS.

Если у одного блока высота задана явно:

- она меньше самого высокого соседа — изменится только этот блок;

- она больше — вырастет контейнер, и остальные элементы пересчитают высоту.

При других значениях **align-items** высота элемента определяется контентом, `padding` и `border`; меняется только положение по поперечной оси.

### Самостоятельное задание

Создайте flex-контейнер с несколькими текстовыми блоками. Включите выравнивание по базовой линии, меняйте длину текста и `font-size` — так проще почувствовать, как работает **baseline**.
