# Emmet

В *Visual Studio Code* плагин *Emmet* уже встроен — вы могли заметить его в прошлом уроке при создании базовой HTML-структуры. В других редакторах *Emmet* ставится отдельно; для популярных сред инструкции найти несложно.

Зачем он нужен? Представьте карточку товара в интернет-магазине: обёртка, название, изображение, цена, кнопка «Заказать». Разместим карточку в секции с классом `cards`. В HTML это выглядит так:

```
<section class="cards">
  <article class="card">
    <div class="card-title">Название товара</div>
    <img src="/images/card-item-1.png" alt="Товар-1">
    <div class="card-price">
      <span class="price-sum">24 000</span><span class="price-cur">руб.</span>
    </div>
    <button>Заказать</button>
  </article>
</section>
```

Разметка несложная, но элементов много. Если набирать её вручную, уйдёт время на теги, атрибуты и закрывающие скобки — хотя структура уже ясна в голове.

Ту же карточку можно получить одной строкой *Emmet*. Сначала она кажется пугающей, но к концу урока каждый символ станет понятен:

> `section.cards>article.card>div.card-title{Название товара}+img[src="/images/card-item-1.png"][alt="Товар-1"]+(.card-price>span.price-sum{24 000}+span.price-cur{руб.})+button{Заказать}`

*Emmet* разворачивает эту запись в HTML выше. Обычно достаточно нажать `Tab` после строки или вызвать команду развёртывания в редакторе.

189 символов вместо 877 — заметная экономия. На больших блоках разница только растёт: одной строкой *Emmet* иногда собирают целую страницу. Инструмент снимает рутину с разметки. Разберём строку с начала:

- `section.cards` — как в CSS: элемент `<section>` с классом `cards`.

- `>` — вложенность. Всё после `section.cards>` попадёт внутрь этой секции.

- `article.card` — `<article class="card">`.

- `div.card-title` — блок с классом `card-title`. Если тег не указан, *Emmet* по умолчанию создаёт `<div>`, поэтому можно писать `.card-title`.

- `{Название товара}` — текст внутри элемента.

- `+` — соседний элемент на том же уровне, а не внутри (в отличие от `>`).

- `img[src="/images/card-item-1.png"][alt="Товар-1"]` — тег `<img>` с атрибутами в квадратных скобках. Можно писать и в одних скобках: `img[src="/images/card-item-1.png" alt="Товар-1"]`.

- `()` — группировка: содержимое скобок идёт рядом с предыдущим элементом.

Дальше — те же приёмы. Ещё примеры:

---

`header>.logo+nav>ul>li*3`

```
<header>
  <div class="logo"></div>
  <nav>
    <ul>
      <li></li>
      <li></li>
      <li></li>
    </ul>
  </nav>
</header>
```

Конструкция `*3` — умножение: повторить элемент три раза.

`header+main+footer`

```
<header></header>
<main></main>
<footer></footer>
```

Быстрый каркас страницы.

---

`div.item-$*3`

```
<div class="item-1"></div>
<div class="item-2"></div>
<div class="item-3"></div>
```

`$` — счётчик при развёртывании (1, 2, 3…). Дополнительно:

- Несколько `$` подряд задают разряды: `div.item-$$$*3` даст `item-001`, `item-002`…

- Стартовое число — через `@`: `div.item-$@5*3` начнёт с пяти.

Счётчик работает и в тексте:

`div.item{Работа номер $}*3`

```
<div class="item">Работа номер 1</div>
<div class="item">Работа номер 2</div>
<div class="item">Работа номер 3</div>
```

Этого хватит надолго. *Emmet* умеет и CSS-сокращения: в `.css`-файле автодополнение VS Code часто опирается на те же правила.
