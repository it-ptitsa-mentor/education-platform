# Глобальный объект Window

В этом уроке разберём `window` — глобальный объект, который браузер отдаёт скриптам.

Через него управляют вкладками и окном: открыть новую вкладку, прокрутить страницу, узнать размеры области просмотра:

```
// Открывает новую вкладку
window.open()
```

`window` виден в консоли DevTools — попробуйте вызвать там метод из примера.

Ещё несколько типичных обращений:

```
// Возвращает объект, содержащий информацию об экране
window.screen
// Screen {availWidth: 1280, availHeight: 775, width: 1280, height: 800, …}

// Перемещает страницу до точки (x-coord, y-coord)
window.scrollTo(0, 1000)

// Видимые высота и ширина страницы
// Меняются при изменении размеров окна
window.innerHeight
window.innerWidth
```

Внутри `window` лежит `document` — с ним мы работаем в следующих уроках при изменении содержимого страницы.

`window` задаёт глобальную область видимости. В нём живут все глобально доступные объекты и функции. Когда вызывают `alert()` или `console.log()` без префикса, движок ищет их в `window` — фактически это `window.alert()`.

То же для любых встроенных API без импорта:

```
console.log('hey')
// window.console.log('hey');

Math.abs(5)
// window.Math.abs(5);

// Можно даже так
close()
// вместо window.close()
```

## Опасность глобального состояния

На `window` как на хранилище состояния в прикладном коде лучше не опираться — это деталь среды выполнения.

Пример:

```
window.globalProperty = 'Global variables are evil'
```

Свойство на `window` становится доступным из любого скрипта на странице — по сути это **глобальная переменная**.

Такие переменные усложняют сопровождение: неясно, кто их пишет и откуда они читаются. На них нельзя надёжно опереться — чужое присваивание ломает логику.

На одной странице часто живут несвязанные скрипты: аналитика, виджеты, реклама. У всех один и тот же `window`. Запись в глобал в одном файле может перетереть имя, которое использует другой.

В аккуратном коде `window` явно почти не встречается, но понимать его роль полезно — так яснее, как JS устроен в браузере.
