# Введение

Современные сайты сильно интерактивны: страница редко перезагружается целиком, контент меняется на лету.

Приложения в браузере стали сложными — от редакторов уровня офисных пакетов до полноценных IDE. Браузерные игры — отдельная индустрия; с аппаратным ускорением они не уступают десктопным.

Без JavaScript этого бы не было.

Изначально язык добавлял мелкие эффекты вроде «снежинок» на странице. Сейчас это профессиональный инструмент. JavaScript — единственный язык, который браузеры исполняют нативно.

Одного языка мало: нужен API для управления страницей и самим браузером. Большая часть возможностей описана в спецификациях HTML5 и связанных стандартов. Например:

- изменение содержимого и структуры страницы

- управление внешним видом

- реакция на действия пользователя

- работа с cookie

- адресная строка, навигация, история

- обмен данными с сервером

- воспроизведение видео

- ввод и вывод данных

- файловые системы и сеть (в пределах браузерных API)

- рисование 2D/3D

С точки зрения кода это в основном глобальные объекты. Центральный из них — **DOM-дерево**.

В этом блоке разберём подключение JavaScript на сайте, базовые способы менять страницу, полифилы, первый Ajax-запрос и событийную модель.
