# Представление символов и метасимвол

Регулярные выражения — формальный язык для поиска и замены фрагментов текста.

Шаблон состоит из:

- Обычных символов, которые нужно найти буквально

- Специальных (мета-)символов с особыми правилами

В примерах темы:

- Сверху — шаблон между разделителями `/`

- Снизу — строка, в которой ищем совпадения

Ищем буквальный текст `java`:

---

/`java`/

`java` \ python ruby1.9 `java`script c#

---

Символ `/` — **разделитель** (*delimiter*). Он отделяет regex от остального текста в записи.

Совпали два вхождения `java` — это **точное** совпадение без метасимволов.

Добавим символ — совпадений не будет:

---

/`javab`/

java \ python ruby1.9 javascript c#

---

**Метасимвол** `.` означает «любой символ»:

---

/`.`/

`java \ python ruby1.9 javascript c#`

---

Подсветилась вся строка.

Комбинация `.y` — любой символ + `y`:

---

/`.y`/

java \ `py`thon ru`by`1.9 javascript c#

---

При смешении мета- и обычных символов важно помнить правила метасимволов.

Шаблон `1.9` — не буквальная точка, а «1», любой символ, «9»:

---

/`1.9`/

java \ python ruby`1.9` javascript c#

java \ python ruby`189` javascript c#

java \ python ruby`1k9` javascript c#

---

Чтобы точка была знаком препинания, её **экранируют** обратным слэшем `\`:

---

/`1\.9`/

java \ python ruby`1.9` javascript c#

java \ python ruby1d9 javascript c#

---

Так же экранируют сам `\`, если он нужен как обычный символ:

---

/`\\`/

java `\` python ruby1d9 javascript c#

---
