# Заключение

Мы подошли к завершению курса «JS: React». Вы познакомились с синтаксисом JSX, научились создавать компоненты, познакомились с жизненным циклом компонента. Этих знаний достаточно, чтобы создавать простые приложения.

Для создания более сложных приложений нужно научиться правильно организовывать состояние. Экосистема реакта содержит различные библиотеки, решающие эту задачу. Одна из самых популярных таких библиотек изучается в курсе [React: Redux Toolkit.](https://buildin.ai/cfe21cf1-2d29-4fd5-a110-46e114189b21) В этом курсе изучаются подходы в организации состояния и работы с ним, курс раскрывает подходы для создания сложных приложений на реакте и подойдет для тех, кто хочет ближе познакомиться с построением архитектуры react-приложений.
