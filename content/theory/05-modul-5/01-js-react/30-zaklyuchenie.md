# Заключение

Курс «JS: React» завершён. Вы разобрали JSX, компоненты и жизненный цикл — этого хватает для простых приложений.

Для более сложных проектов важно правильно организовать состояние. В экосистеме React для этого есть библиотеки; одна из популярных — Redux Toolkit, она разбирается в следующем курсе [React: Redux Toolkit.](https://buildin.ai/cfe21cf1-2d29-4fd5-a110-46e114189b21) Там — паттерны хранения и обновления state и архитектура крупных React-приложений.
