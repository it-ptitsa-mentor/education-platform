# Навигация

В графической ОС после загрузки вы попадаете на **рабочий стол** — точку входа, откуда через меню запускаются задачи. Для файлов обычно открывают проводник.

В командной строке иначе: после старта система ждёт команду. Режим привязан к файловой системе — вы всегда находитесь в какой-то директории, её называют **рабочей** (*working directory*).

В этом уроке — навигация по директориям через оболочку.

## Как проверить, в какой директории мы находимся

Текущую директорию показывает команда `pwd`:

```
pwd

/Users/guest
```

Имя `pwd` — сокращение от *print working directory*. Так устроены многие команды: короткое имя легче запомнить.

Новая сессия обычно открывается в домашней директории пользователя. У каждого пользователя свой путь — у вас он будет другим.

В примере две неожиданности для привыкших к Windows:

1. В начале не буква диска, а корень `/` — вершина всей файловой системы

1. Разделитель — прямой слэш `/`, а не обратный `\`

Подробнее о структуре — в следующих уроках; сейчас сфокусируемся на навигации.

## Как посмотреть список файлов

Команда `ls` (от *list*) выводит файлы и подкаталоги текущей директории:

```
ls

Desktop  Documents Downloads Library  Movies  Music  Pictures Public
```

## Как переместиться в другую директорию

Команда `cd` (*change directory*) меняет рабочую директорию. Ей передают **аргумент** — каталог, куда нужно перейти:

```
# Входим в директорию
cd Music
# Смотрим ее содержимое
ls

iTunes
```

```
# Смотрим текущую рабочую директорию
pwd

/Users/guest/Music
```

```
# Если имя директории содержит пробел, то его нужно экранировать с помощью `\`
cd Best\ music
```

Остановимся подробнее. Путь к файлу бывает:

- **Абсолютный** — от корня файловой системы

- **Относительный** — от текущей рабочей директории

Выше был относительный путь. Отличить просто:

- Абсолютный начинается с `/`

- Относительный — во всех остальных случаях

При относительном пути `cd` вычисляет абсолютный: берёт текущую директорию */Users/guest/* и дописывает *Music* → */Users/guest/Music*.

`cd` принимает оба вида путей:

```
# Неважно, в каком месте
cd /Users/guest/Music # Абсолютный путь
```

Абсолютный путь однозначно указывает объект в ФС; относительный — только вместе с текущей директорией. Абсолютный путь работает из любого места: если каталог существует, переход произойдёт.

Допустим, вы в */Users/guest/Music* и хотите вернуться в */Users/guest*. Можно указать абсолютный путь:

```
cd /Users/guest
```

Проще — специальное имя `..` (на уровень вверх):

```
# В директории /Users/guest/Music
cd ..
pwd

/Users/guest
```

Через разделитель можно подняться на несколько уровней:

```
# В директории /Users/guest/Music
# Выходим на два уровня вверх
cd ../..
pwd

/Users
```

**Одинарная точка** `.` означает текущую директорию: `cd Music` и `cd ./Music` эквивалентны.

Третий способ в домашний каталог — `cd` без аргументов:

```
# Из любого места
cd
pwd

/Users/guest
```

Четвёртый — тильда `~` ([тильда](https://ru.wikipedia.org/wiki/%D0%A2%D0%B8%D0%BB%D1%8C%D0%B4%D0%B0)): при выполнении подставляется абсолютный путь домашней директории:

```
# Из любого места
cd ~/Music
pwd

/Users/guest/Music
```

Если вы дома и хотите список файлов в *Music*, можно зайти в каталог и вызвать `ls`.

Есть и другой путь: `ls` принимает **аргумент** — каталог для просмотра:

```
ls Music

iTunes
```

Как и у `cd`, к аргументу `ls` применимы абсолютные и относительные пути — это общее правило для команд с путями.

`cd -` возвращает в предыдущую директорию; повторные вызовы переключают между двумя последними каталогами.

`cd`, `ls` и `pwd` — база навигации. Их достаточно, чтобы не потеряться.

Со временем набирать длинные пути лениво — нажмите `Tab` дважды и воспользуйтесь **автодополнением** имён.

---

### Самостоятельная работа

Изучите каталоги своей файловой системы. При переходах `cd` используйте `Tab` для автозаполнения.
