# Вложенные массивы

Элементом массива может быть что угодно, в том числе другой массив:

```
const arr1 = [[3]]
arr1.length // 1

const arr2 = [1, [3, 2], [3, [4]]]
arr2.length // 3
```

Вложенный массив — один элемент «сверху»; отсюда `length` у второго примера равен 3. Элементы можно записывать построчно:

```
const arr2 = [
  1, // первый элемент (число)
  [3, 2], // второй элемент (массив)
  [3, [4]], // третий элемент (массив)
]
arr2.length // 3
```

Вложенность не ограничена.

Доступ к вложенным элементам — цепочка индексов:

```
const arr1 = [[3]]
arr1[0][0] // 3

const arr2 = [1, [3, 2], [3, [4]]]
arr2[2][1][0] // 4
```

Сначала может казаться непривычным, но это тренировка:

```
const arr2 = [
  1,
  [3, 2],
  [3, [4]],
]

arr2[2] // [3, [4]]
arr2[2][1] // [4]
arr2[2][1][0] // 4
```

Изменение и добавление:

```
const arr1 = [[3]]
arr1[0] = [2, 10]
arr1.push([3, 4, 5])

// [[2, 10], [3, 4, 5]]
```

Вложенные массивы меняют по индексу:

```
const arr1 = [[3]]
arr1[0][0] = 5
// [[5]]
```

И добавляют элементы во вложенный массив:

```
const arr1 = [[3]]
arr1[0].push(10)
// [[3, 10]]
```

Зачем это нужно? Матрицы, игровые поля — классический пример крестиков-ноликов.

Задача: поле задано массивом массивов; проверить, есть ли на поле хотя бы один заданный символ (`x` или `o`).

```
// Инициализируем поле
const field = [
  [null, null, null],
  [null, null, null],
  [null, null, null],
]

// Делаем ход:
field[1][2] = 'x'
// [
//     [null, null, null],
//     [null, null, 'x'],
//     [null, null, null],
// ]
```

Функция проверки:

```
const hasPlayerMoved = (field, symbol) => {
  // Обходим поле. Каждый элемент — это строчка в игровом поле.
  for (const row of field) {
    // метод includes проверяет присутствует ли элемент в массиве,
    if (row.includes(symbol)) { // Если присутствует, значит мы нашли то, что искали.
      return true
    }
  }

  // Если поле было просмотрено, но ничего не нашли,
  // значит ходов не было.
  return false
}
```

Проверка:

```
hasPlayerMoved(field, 'x') // true
hasPlayerMoved(field, 'o') // false
```
