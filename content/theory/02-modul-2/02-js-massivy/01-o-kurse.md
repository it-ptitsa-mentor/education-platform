# О курсе

Программирование по-настоящему оживает, когда появляется работа с наборами элементов — коллекциями. Вот лишь часть типичных сценариев:

- Постраничный вывод данных на сайте

- Подсчёт итоговой суммы заказа по каждой позиции

- Списки друзей, сообщений, фильмов и тому подобное

- Обработка набора DOM-узлов (HTML, фронтенд-разработка)

Любой список в реальном или цифровом мире — с точки зрения кода это коллекция элементов. В JavaScript для таких наборов служит *массив*: структура данных, с которой можно работать как с единым целым.

```javascript
// Определение массива друзей
const friends = ['petya', 'vasya', 'ivan']
```

В отличие от примитивных типов, массивы в JavaScript изменяемы: и по содержимому, и по длине. Это расширяет возможности, но добавляет ответственности. Одну задачу с массивами можно решить множеством способов; далеко не все из них удачные — часть окажется медленной, запутанной и трудной для отладки.

Поэтому теме массивов посвящён не пара уроков, а целый развёрнутый курс. В нём разбираются типичные ситуации, где массивы — естественный инструмент. Полученные здесь знания станут опорой для всей дальнейшей разработки. Основные темы курса:

- Манипуляции с массивами

- Обработка массивов в циклах

- Работа с вложенными массивами, используя вложенные циклы

- Сортировка массивов

- Работа со строками через массивы

Помимо массивов, затронем алгоритмы и структуры данных: познакомитесь с алгоритмической сложностью, реализуете несколько классических алгоритмов, которые часто встречаются на собеседованиях. Базовое понимание этих тем критично для эффективного кода.
