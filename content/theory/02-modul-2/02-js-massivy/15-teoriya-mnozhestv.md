# Теория множеств

Теория множеств важна разработчикам: данные в программах часто ведут себя как множества, и к ним применимы пересечение, объединение и другие операции.

Полный курс математики не нужен — достаточно базовых понятий и нескольких операций. Концепции интуитивны и хорошо стыкуются с здравым смыслом.

## Краткая терминология

**Множество** — набор объектов, рассматриваемый как целое. Пример: все цифры от 0 до 9 — десять элементов.

Не любой набор — множество: элементы должны быть **уникальными**. *1, 1, 3* — не множество; *1, 3, 5* — множество.

Множества связаны отношениями: натуральные числа — подмножество целых, целые — рациональных и т.д. **Подмножество** — все элементы одного множества входят в другое (**надмножество**).

Представление множеств кружками довольно удобно. Можно быстро оценить как друг с другом соотносятся разные множества.

Объекты множеств — не только числа: люди на остановке, жильцы дома, города.

В коде множествами часто выступают массивы и таблицы БД. В JavaScript есть [Set](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Set); для него нужны ООП-концепции из более поздних курсов.

## Операции над множествами

Множества полезны, когда с ними что-то делают. В Facebook блок «общие друзья» — пересечение двух множеств друзей.

Пересечение и родственные операции встречаются повсюду. Результат — снова множество с теми же правилами (в т.ч. уникальность элементов).

### Пересечение

Пересечение — элементы, общие **всем** исходным множествам.

```
// Друзья одного человека
const friends1 = ['vasya', 'kolya', 'petya']

// Друзья другого человека
const friends2 = ['igor', 'petya', 'sergey', 'vasya', 'sasha']

// Общие друзья
// Эта функция принимает любое количество массивов.
// То есть вы можете находить пересечение любого количества массивов за один вызов.
_.intersection(friends1, friends2) // ['vasya', 'petya']
```

### Объединение

Объединение — все элементы из всех множеств:

```
const friends1 = ['vasya', 'kolya', 'petya']
const friends2 = ['igor', 'petya', 'sergey', 'vasya', 'sasha']

_.union(friends1, friends2) // ['vasya', 'kolya', 'petya', 'igor', 'sergey', 'sasha']
```

Каждый друг встречается один раз.

### Дополнение (разность)

Разность — элементы первого множества, которых нет во втором. В коде часто *diff*:

```
const friends1 = ['vasya', 'kolya', 'petya']
const friends2 = ['igor', 'petya', 'sergey', 'vasya', 'sasha']

_.difference(friends1, friends2) // ['kolya']
```

### Принадлежность множеству

Проверка через `includes()`:

```
const terribleNumbers = [4, 13]

if (terribleNumbers.includes(10)) {
  console.log('woah!')
}
```
