# Установка JavaScript

Начнём с установки **Node.js** и знакомства с **REPL**.

Если Node.js ещё не установлен, поставьте его по [официальной инструкции](https://nodejs.org/en/download/package-manager). После установки перезагрузите компьютер.

Проверим, что Node.js на месте. В терминале выполните `node --version`. Вывод должен быть примерно таким:

```
# Версия, которую вы увидите, должна быть не ниже, чем показана здесь
# Иначе возможно появление ошибок

node --version
v17.6
```

## Интерактивный запуск кода

Запустите JavaScript так:

```
node
>
```

Команда *node* без аргументов открывает REPL — интерактивную оболочку, которая:

- Ожидает ввод от пользователя (*Read*)

- Выполняет введенный код (*Eval*)

- Печатает на экран результат (*Print*)

- Затем снова входит в режим ожидания (*Loop*)

REPL похож на терминал, только вместо shell-команд вы вводите JavaScript. Можно набрать строку и сразу выполнить её по `Enter`:

```
node
> 1 + 5
6 # Результат выполнения сразу отображается на экране
> 7 % 2
1
```

Так удобно проверять гипотезы, отлаживать фрагменты и считать в уме.

REPL помнит переменные и предыдущий ввод:

```
node
> a = 5
5
> b = 10
10
> a + b
15
```

Выйти из REPL можно так:

1. Наберите команду `.exit` и нажмите `Enter`

1. Два раза нажмите `CTRL + C`

1. Нажмите `CTRL + D`

После этого вы снова в обычной командной оболочке:

```
node
Welcome to Node.js v16.4.1.
Type ".help" for more information.
> 1 + 5
6
>
(To exit, press ^C again or ^D or type .exit)
>
$
```

Различайте **REPL Node.js** и **командную оболочку**. Команда *node* запускает отдельную программу; она не совпадает с оболочкой, из которой вы её вызвали.

Если ввести в REPL команду оболочки, получите синтаксическую ошибку:

```
node
> cd ~
cd ~
   ^

SyntaxError: Unexpected token ~
```

## Запуск кода из файлов

Для полноценной разработки REPL быстро становится тесным. На практике код пишут в текстовых файлах.

Популярный редактор — [VS Code](https://code.visualstudio.com/). Установите его и осмотрите интерфейс; возможности расширяются плагинами.

Открыть VS Code из терминала: `code` и путь к проекту. Например, текущая директория:

```
code .
```

Создайте файл *index.js* с содержимым:

```
console.log('Hello, JS!')
```

В VS Code есть встроенный запуск кода, но пока учитесь — запускайте вручную.

В терминале в той же директории, где лежит файл:

```
# Сначала перейдите в директорию с файлом index.js
node index.js
'Hello, JS!'
```

## Самостоятельная работа

1. Настройте вашу операционную систему, так чтобы она была готова к работе с Node.js

1. Установите Node.js

1. Запустите репл и попробуйте выполнить внутри JavaScript код

1. Вычислите в репле значение выражения `Math.sqrt(256) + 100`

1. В проекте my*-js* создайте файл *index.js* и добавьте туда `console.log('Hello, JS!');`

1. Запустите этот файл командой *node index.js*, убедитесь что на экран вывелась строчка *Hello, JS!*
