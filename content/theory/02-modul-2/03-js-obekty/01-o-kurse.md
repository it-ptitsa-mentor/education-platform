# О курсе

Объект — один из самых гибких и выразительных типов в JavaScript. Им удобно описывать сущности из реального мира: например, пользователя учебной платформы с email, именем, паролем и прочими полями — всё это собирается в одну структуру.

```javascript
// Синтаксис разбирается в следующем уроке
const user = {
  email: 'inna@example.com',
  name: 'Inna',
  password: 'qwerty',
}
```

Содержимое файла *package.json* тоже можно представить в виде объекта JavaScript. Более того, название формата JSON расшифровывается как *JavaScript Object Notation*:

```javascript
// json это представление объекта в виде текста
const data = {
  name: 'MyProjectName',
  description: 'My Projects\'s Description',
  dependencies: [
    // тут зависимости
  ],
  // тут остальные свойства
}
```

И, наконец, в это сложно поверить, но функции и массивы в JavaScript тоже являются объектами.

```javascript
typeof [] // object
// С функцией чуть сложнее
// Проверяем, является ли функция объектом
Math.random instanceof Object // true
```

Объекты пронизывают JavaScript-разработку целиком. Их применяют и как ассоциативные массивы (хранилище пар ключ-значение), и как сущности в смысле объектно-ориентированного программирования — с этим мы познакомимся позже.
