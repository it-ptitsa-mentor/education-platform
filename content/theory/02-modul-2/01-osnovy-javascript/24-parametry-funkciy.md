# Параметры функций

<!-- IMG (из Buildin, перезалить отдельно) -->
Функции принимают **параметры** (аргументы при вызове) и возвращают значения. С параметрами вы уже работали:

```
// Принимает на вход параметр любого типа
console.log('я параметр')
// Принимает на вход два строковых параметра
// первый – что ищем, второй – на что меняем
'google'.replace('go', 'mo') // moogle;
// Принимает на вход два числовых параметра
// первый – начальный индекс (включая), второй – конечный индекс (не включая)
'world'.slice(1, 3) // or
```

Научимся объявлять параметры у своих функций. Задача: `getLastChar()` — вернуть последний символ переданной строки. Использование:

```
// Передача параметров напрямую без переменных
getLastChar('World') // d
// Передача параметров через переменные
const name1 = 'World'
getLastChar(name1) // d
const name2 = 'Goo'
getLastChar(name2) // o
```

Из примеров следует:

- Нам нужно определить функцию `getLastChar()`

- Функция должна принимать на вход один параметр строкового типа

- Функция должна возвращать значение строкового типа

Определение функции:

```
const getLastChar = (text) => {
  // Вычисляем индекс последнего символа как длина строки - 1
  return text[text.length - 1]
}
```

В скобках — имя параметра (`text`). Имя любое, но должно отражать смысл. Эквивалент с другим именем:

```
const getLastChar = (str) => {
  return str[str.length - 1]
}
```

Значение параметра задаётся при каждом вызове.

```
// Внутри функции str будет равна 'world'
getLastChar('world') // d

// Внутри функции str будет равна 'code'
getLastChar('code') // e

// Внутри функции str будет равна 'Winter is coming'
// Имя переменной снаружи не связано с именем переменной в определении функции
const text = 'Winter is coming'
getLastChar(text) // g
```

Несколько параметров перечисляют через запятую:

```
// функция по нахождению среднего числа
const average = (a, b) => {
  return (a + b) / 2
}

average(1, 5) // 3
average(1, 2) // 1.5
```

У методов та же схема — столько параметров, сколько нужно операции:

```
// первый параметр – что ищем
// второй параметр – на что меняем
'google'.replace('go', 'mo') // moogle
```

В определении перечисляют параметры через запятую с понятными именами. Условная `replace()` для замены подстроки:

```
const replace = (text, from, to) => {
  // здесь тело функции, но мы его
  // опускаем, чтобы не отвлекаться
}

replace('google', 'go', 'mo') // moogle
```

При двух и более параметрах важен **порядок** аргументов — перестановка меняет смысл:

```
// ничего не заменилось,
// так как внутри google нет mo
replace('google', 'mo', 'go') // google
```
