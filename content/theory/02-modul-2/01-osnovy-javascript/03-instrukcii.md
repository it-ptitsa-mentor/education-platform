# Инструкции

**Инструкция** — команда интерпретатору. Код на JavaScript — последовательность инструкций; обычно их разделяют `;`.

<!-- IMG (из Buildin, перезалить отдельно) -->
Пример с двумя инструкциями:

```javascript
console.log('Mother of Dragons.')
console.log('Dracarys!')
// => Mother of Dragons.
// => Dracarys!
```

Технически можно писать несколько инструкций в одной строке:

```javascript
console.log('Mother of Dragons.'); console.log('Drakarys!')
```

Поведение то же, но читать так трудно — поэтому инструкции пишут с новой строки.

Инструкция — минимальная единица выполнения: интерпретатор идёт сверху вниз. Разработчику важно видеть этот порядок и делить программу на части для анализа.
