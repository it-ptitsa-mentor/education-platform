# Точки на координатной плоскости

Геометрия — удобная тема для тренировки моделирования: она наглядна и интуитивна. Напомним базовые понятия.

**Координатная (декартова) плоскость** — плоскость с системой координат на двух перпендикулярных осях `x` и `y`.

```
                     6 | y
                     5 |
                     4 |
                     3 |     . (2, 3)
                     2 |
                     1 |
                       |
----------------------------------------------------
                       |  1  2  3  4  5  6         x
                       |
                       |
                       |
                       |
                       |
                       |
```

Простейший примитив — **точка**. Её положение задаётся двумя координатами; в математике пишут `(2, 3)` — сначала `x`, затем `y`. В коде точку можно хранить массивом из двух чисел.

```
const point = [2, 3]
```

Этого хватает для полезных действий. Чтобы найти точку, симметричную относительно оси `x`, меняют знак у координаты `y`.

```
                     6 | y
                     5 |
                     4 |
                     3 |     . (2, 3)
                     2 |
                     1 |
                       |
----------------------------------------------------
                       |  1  2  3  4  5  6         x
                    -1 |
                    -2 |
                    -3 |     . (2, -3)
                    -4 |
                    -5 |
                    -6 |
```

```
const point = [2, 3]

const [x, y] = point
// x = 2, y = 3

const symmetricalPoint = [x, -y]
// x = 2, y = -3
```

Иногда нужна точка посередине отрезка между двумя точками. Координаты — среднее арифметическое: `x` равно `(x1 + x2) / 2`, `y` — `(y1 + y2) / 2`.

```
const getMiddlePoint = (p1, p2) => {
  const x = (p1[0] + p2[0]) / 2
  const y = (p1[1] + p2[1]) / 2

  return [x, y]
}

const point1 = [2, 3]
const point2 = [-4, 0]

console.log(getMiddlePoint(point1, point2)) // => [-1, 1.5]
```

Подобных операций в геометрии много. Функции для точек логично собрать в модуль `points`.

Точки объединяются в **отрезки**. Отрезок задаётся парой точек — началом и концом. В коде его тоже можно представить массивом из двух точек.

```
const point1 = [3, 4]
const point2 = [-8, 10]
const segment = [point1, point2]
```
