# Теория: О курсе

В любом приложении центральны данные: их получают, обрабатывают, хранят и показывают пользователю. Часть данных проста (строка с текстом, массив сообщений), часть сложна — профиль пользователя с десятками полей.

```
// Очень упрощенно, в реальности все сложнее
const student = {
  name: 'Petya',
  encryptedPassword: ...,
  facebookId: ...,
  payments: [/* платежная информация */],
  currentGroup: ...,
  finishedCourses: [/* список пройденных курсов */]
};
```

Один подход — процедурные функции над данными (смена пароля и т.д.):

```
// Внутри себя делает шифрование и обновление user
// user меняется внутри функций
changePassword(user, 'new secret password')
addFinishedCourse(user, finishedCourse)
```

Другой — методы на самих объектах:

```
user.changePassword('new secret password')
user.addFinishedCourse(course)
```

Когда программа — набор объектов с поведением (методами), это объектно-ориентированное программирование (ООП). «Объект» здесь не только тип `object`, а сущность с поведением.

## Изучение ООП

ООП в JavaScript — большая сквозная тема; ей посвящено несколько курсов — от синтаксиса до организации кода.

Этот курс даёт базовые концепции и синтаксис и затрагивает особенности JS среди ООП-языков. Дальше — принципы построения программ и внутренности языка (прототипы и др.).

Основные темы курса:

- Объекты

- Классы

- Инкапсуляция

- Сокрытие данных (Data hiding)

- Исключения

Эти темы важны с первых дней работы на JS-проекте, хотя уверенно применять их получается не сразу. К ООП мы ещё вернёмся в следующих курсах — углубим понимание, конструкторы и прототипы.
