# Объекты первого класса

В языках программирования говорят об **объектах первого рода (класса)** — значениях, которые можно передать в функцию, вернуть из неё и присвоить переменной или константе. Сюда относятся числа, строки, массивы, логические значения и другие данные.

```
// 5 — число, объект первого рода (сохранено в константе)
const num = 5

// 2 — число, объект первого рода (аргумент функции)
// содержимое константы num — объект первого рода (аргумент функции)
// содержимое константы result — объект первого рода (возвращаемое значение)
const result = Math.pow(num, 2)
```

Первоклассными могут быть не только «данные», но и конструкции языка — в том числе **функции**. В JavaScript функции — объекты первого рода; от этого сильно зависит стиль кода. Ниже — как с функциями обращаются как с данными.

## Сохранение в константе

```
// Функция записывается в константу!
const x = () => console.log('I love JS')

x() // => 'I love JS'
```

Здесь два шага:

1. Определение функции: `() => console.log('I love JS')`

1. Константа `x` и сохранение в неё функции: `const x =`

Минимальное определение — `() => {}`: пустая функция. Присваивать её константе или нет — отдельный вопрос.

Функцию из `x` можно передать в другую константу. Копируется не «сама функция», а ссылка на неё:

```
const a = () => console.log('I love JS')

a() // => 'I love JS'

const b = a

b() // => 'I love JS'
```

Функцию можно вызвать без сохранения в константу:

```
(() => console.log('I love JS'))() // => I love JS
```

Создали функцию и сразу вызвали оператором `()`. Определение в скобках, чтобы интерпретатор видел границы. После вызова доступ к функции теряется — она нигде не сохранена.

Имя у константы есть, у самой функции — нет: это **анонимная** функция (в других языках — лямбда).

## Создание внутри другой функции

Анонимная функция — выражение; её можно объявить там, где допустимы выражения, в том числе внутри другой функции:

```
const sum = (a, b) => {
  // определили "внутреннюю" анонимную функцию и
  // сохранили в константе innerSum
  const innerSum = (x, y) => x + y

  // вызвали внутреннюю функцию и
  // вернули результат вызова наружу из sum
  return innerSum(a, b)
}

sum(1, 4) // 5
```

Определение функции — обычное выражение: его передают аргументом и возвращают как значение. Подробнее — в уроке о **функциях высшего порядка**.

Анонимные функции сильно расширяют выразительность JS. Функции, которые создают, возвращают и принимают функции — обычный способ писать на JavaScript.
